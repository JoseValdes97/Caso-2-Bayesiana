Obtendremos una tabla con los estadsticos necesarios por dominio para trabajar con las distribuciones condicionales completas y tener una idea de los datos.

  1. la funcin n() solo funciona dentro de una funcion actual como summarise en el estado actual y devuelve el tamao del grupo
# tabla con los estadisticos
Estadisticos <- Data %>% 
      group_by(Dominio) %>% 
      summarise(Dominio = unique(Dominio), nj = n(), yb = mean(Ingtot),
                s2 = var(Ingtot))
